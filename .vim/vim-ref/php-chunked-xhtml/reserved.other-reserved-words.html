その他の予約語の一覧

これらの名前は、クラスやインターフェイスそしてトレイトの名前として使えません。 また、名前空間の中であっても使うことはできません。 この中のいくつかは、今後の PHP に追加されるであろう機能のために予約されています。 そのときに過去との互換性を維持するようにするためです。

予約語
int (PHP 7 以降) float (PHP 7 以降) bool (PHP 7 以降) string (PHP 7 以降)
true (PHP 7 以降) false (PHP 7 以降) null (PHP 7 以降) resource (PHP 7 以降)
object (PHP 7 以降) mixed (PHP 7 以降) numeric (PHP 7 以降)
