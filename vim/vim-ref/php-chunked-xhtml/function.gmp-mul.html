gmp_mul

(PHP 4 >= 4.0.4, PHP 5)

gmp_mul — 数値を乗算する

説明

GMP gmp_mul ( GMP $a , GMP $b )

a と b をかけ、 結果を返します。

パラメータ

a

b を掛けられる数。

PHP 5.5 以前での GMP 数リソース、PHP 5.6 以降での GMP オブジェクト、あるいは数値に変換可能な数値形式の文字列。

b

a に掛ける数。

PHP 5.5 以前での GMP 数リソース、PHP 5.6 以降での GMP オブジェクト、あるいは数値に変換可能な数値形式の文字列。

返り値

PHP 5.5 以前は GMP 数リソースを、そして PHP 5.6 以降は GMP オブジェクトを返します。

例

例1 gmp_mul() の例

<?php
$mul = gmp_mul("12345678", "2000");
echo gmp_strval($mul) . "\n";
?>

上の例の出力は以下となります。

24691356000
