$HTTP_RAW_POST_DATA

(PHP 4, PHP 5)

$HTTP_RAW_POST_DATA — 生の POST データ

説明

警告

この機能は PHP 5.6.0 で 非推奨になります。この機能に頼らないことを強く推奨します。

$HTTP_RAW_POST_DATA には生の POST データが格納されます。 always_populate_raw_post_data も参照ください。

$HTTP_RAW_POST_DATA の代わりに、 php://input を使うべきです。
