Worker::unstack

(PECL pthreads >= 2.0.0)

Worker::unstack — スタック処理

説明

public integer Worker::unstack ([ Threaded &$work ] )

指定したオブジェクト (パラメータを省略した場合は、すべてのオブジェクト) を、ワーカーのスタックから削除します。

パラメータ

work

事前に Worker に積んだ Threaded オブジェクト。

返り値

スタックの新しい長さを返します。

例

例1 ワーカーのスタックからのオブジェクトの削除

<?php
class Work extends Threaded {
    public function run() {
        
    }
}

$my = new Worker();
$work = new Work();
var_dump($my->stack($work));
var_dump($my->unstack($work));
?>

上の例の出力は以下となります。

int(1)
int(0)
