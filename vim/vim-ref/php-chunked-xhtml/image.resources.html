リソース型

この拡張モジュールでは、以下のリソース型を定義しています。

GD におけるリソース型の一覧
名前 説明 備考
gd 画像リソース。imagecreatefrompng() などの関数が利用します。  
gd font フォントリソース。imageloadfont() が返します。  
gd PS font PostScript Type 1 フォントリソース。imagepsloadfont() が返します。 PHP 7.0.0 で削除されました。
gd PS encoding PostScript Type 1 エンコーディングリソース。imagepsencodefont() が返します。 PHP 7.0.0 で削除されました。
