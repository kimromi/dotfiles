HaruPage::measureText

(PECL haru >= 0.0.1)

HaruPage::measureText — 指定した幅の一行中に配置できる文字のバイト長を計算する

説明

int HaruPage::measureText ( string $text , float $width [, bool $wordwrap = false ] )

指定した幅の一行中に配置できる文字のバイト長を返します。

パラメータ

text

調べたいテキスト。

width

行の長さ。

wordwrap

このパラメータを TRUE に設定すると、ワードラップ処理を "エミュレート" します。行の中に単語全体 (空白文字で区切る) が収まらなくなると、その単語はその行に含めません。

返り値

指定した幅の一行中に配置できる文字のバイト長を返します。 シングルバイトエンコーディングの場合は、文字数と同じになります。 マルチバイトエンコーディングの場合は、必ずしもそうなるとは限りません。

エラー / 例外

エラー時に HaruException をスローします。
