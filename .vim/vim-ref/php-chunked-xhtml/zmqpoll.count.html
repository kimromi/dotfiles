ZMQPoll::count

(PECL zmq >= 0.5.0)

ZMQPoll::count — Count items in the poll set

説明

public integer ZMQPoll::count ( void )

Count the items in the poll set.

パラメータ

この関数にはパラメータはありません。

返り値

Returns an integer representing the amount of items in the poll set.
