json_last_error_msg

(PHP 5 >= 5.5.0, PHP 7)

json_last_error_msg — 直近の json_encode() や json_decode() の呼び出しのエラー文字列を返すl

説明

string json_last_error_msg ( void )

パラメータ

この関数にはパラメータはありません。

返り値

成功した場合にエラーメッセージ、エラーが発生しなかった場合は "No Error"、 失敗した場合に FALSE を返します。
