Threaded::chunk

(PECL pthreads >= 2.0.0)

Threaded::chunk — Manipulation

説明

public array Threaded::chunk ( integer $size , boolean $preserve )

Fetches a chunk of the objects property table of the given size, optionally preserving keys

パラメータ

size

The number of items to fetch

preserve

Preserve the keys of members, by default false

返り値

An array of items from the objects property table

例

例1 Fetch a chunk of the property table

<?php
$safe = new Threaded();

while (count($safe) < 10) {
    $safe[] = count($safe);
}

var_dump($safe->chunk(5));
?>

上の例の出力は以下となります。

array(5) {
  [0]=>
  int(0)
  [1]=>
  int(1)
  [2]=>
  int(2)
  [3]=>
  int(3)
  [4]=>
  int(4)
}
