fann_get_errno

(PECL fann >= 1.0.0)

fann_get_errno — Returns the last error number

説明

int fann_get_errno ( resource $errdat )

Returns the last error number.

パラメータ

errdat

ニューラルネットワークリソースあるいはニューラルネットワークトレーニングリソース。

返り値

The error number, or FALSE on error.

参考

  • fann_reset_errno() - Resets the last error number
  • fann_get_errstr() - Returns the last errstr
