Imagick::queryFormats

(PECL imagick 2.0.0)

Imagick::queryFormats — Imagick がサポートするフォーマットを返す

説明

array Imagick::queryFormats ([ string $pattern = "*" ] )

Imagick がサポートするフォーマットを返します。

パラメータ

pattern

返り値

Imagick がサポートするフォーマットを含む配列を返します。

エラー / 例外

エラー時に ImagickException をスローします。

例

例1 Imagick::queryFormats()

<?php
    function render() {
        $output = "";
        $input = \Imagick::queryformats();
        $columns = 6;

        $output .= "<table border='2'>";

        for ($i=0; $i < count($input); $i += $columns) {
            $output .= "<tr>";
            for ($c=0; $c<$columns; $c++) {
                $output .= "<td>";
                if (($i + $c) <  count($input)) {
                    $output .= $input[$i + $c];
                }
                $output .= "</td>";
            }
            $output .= "</tr>";
        }

        $output .= "</table>";

        return $output;
    }

?>
