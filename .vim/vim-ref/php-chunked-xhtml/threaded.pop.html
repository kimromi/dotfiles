Threaded::pop

(PECL pthreads >= 2.0.0)

Threaded::pop — Manipulation

説明

public boolean Threaded::pop ( void )

Pops an item from the objects property table

返り値

The last item from the objects property table

例

例1 Popping the last item from the property table of a threaded object

<?php
$safe = new Threaded();

while (count($safe) < 10)
    $safe[] = count($safe);

var_dump($safe->pop());
?>

上の例の出力は以下となります。

int(9)
