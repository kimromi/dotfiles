fann_reset_errno

(PECL fann >= 1.0.0)

fann_reset_errno — Resets the last error number

説明

void fann_reset_errno ( resource $errdat )

Resets the last error number.

パラメータ

errdat

ニューラルネットワークリソースあるいはニューラルネットワークトレーニングリソース。

返り値

値を返しません。
