Imagick::shaveImage

(PECL imagick 2.0.0)

Imagick::shaveImage — 画像の輪郭からピクセルを刈り取る

説明

bool Imagick::shaveImage ( int $columns , int $rows )

画像の輪郭からピクセルを刈り取ります。 新しい画像に必要なメモリを確保し、新しい画像へのポインタを返します。

パラメータ

columns

rows

返り値

成功した場合に TRUE を返します。

例

例1 Imagick::shaveImage()

<?php
function shaveImage($imagePath) {
    $imagick = new \Imagick(realpath($imagePath));
    $imagick->shaveImage(100, 50);
    header("Content-Type: image/jpg");
    echo $imagick->getImageBlob();
}

?>
