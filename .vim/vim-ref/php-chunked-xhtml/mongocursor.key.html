MongoCursor::key

(PECL mongo >=0.9.0)

MongoCursor::key — 現在の結果の _id、あるいは結果セット内のインデックスを返す

説明

public string|int MongoCursor::key ( void )

パラメータ

この関数にはパラメータはありません。

返り値

現在の結果の _id を文字列で返します。 結果に _id が含まれない場合は、結果セット内の数値インデックスを表す整数値を返します。
