Imagick::getQuantumRange

(PECL imagick 2.0.0)

Imagick::getQuantumRange — Imagick quantum range を返す

説明

array Imagick::getQuantumRange ( void )

Imagick インスタンスの quantum range を文字列で返します。

返り値

quantum range を連想配列で返します。 integer ("quantumRangeLong") と string ("quantumRangeString") が含まれます。

エラー / 例外

エラー時に ImagickException をスローします。
